# Weather App

Aplicacion del clima desarrollada con HTML, CSS y JavaScript vanilla, utilizando la API de OpenWeather para consultar datos meteorologicos en tiempo real.

## Demo

- Vercel: https://weather-app-liard-rho-65.vercel.app

## Vista previa

![Vista previa](./vistaPrevia.png)

## Funcionalidades

- Busqueda por ciudad
- Filtro opcional por pais
- Temperatura actual y descripcion del clima
- Sensacion termica, humedad, presion y viento
- Mensajes de error claros
- Interfaz responsive

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript vanilla
- OpenWeather API

## Aprendizajes

Este proyecto me permitio practicar:

- Consumo de APIs con `fetch`
- Manejo de asincronia con `async/await`
- Renderizado dinamico en el DOM
- Validacion de entradas
- Diseño de interfaces para proyectos frontend chicos
